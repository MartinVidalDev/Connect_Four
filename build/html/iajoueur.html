Model.IAJoueur

Fonctions permettant de gérer l’intelligence artificielle.

Exemple d’utilisation :

from Model.Constantes import *
from Model.IAJoueur import IA

# Affectation de l'IA
joueur[const.PLACER_PION] = IA
Model.Joueur.IA(joueur: dict) → int

Fonction qui permet de faire jouer l’intelligence artificielle (IA)..

Paramètres:

joueur – Le joueur qui est représenté par l’IA

Renvoie:

La fonction retourne le numéro de colonne où l’IA décide de jouer

Model.Joueur.type_joueur(joueur: dict) → bool

Détermine si le paramètre peut correspondre à un joueur.

Paramètres:

joueur – Paramètre à tester

Renvoie:

True s’il peut correspondre à un joueur, False sinon.

Model.Joueur.PlacerPionJoueur(joueur: dict) → int

Cette fonction permet de choisir une colonne aléatoirement dans mon plateau.

Paramètres:

joueur – Joueur dont on veut tirer aléatoirement la colonne

Renvoie:

Retourne le numéro de la colonne

Lève:

TypeError – Si le premier paramètre n’est pas un pion

Model.Joueur.getPlateauJoueur(joueur: dict) → list

Fonction qui permet d’obtenir le plateau du joueur passé en paramètre.

Paramètres:

joueur – Joueur dont on cherche le plateau

Renvoie:

La fonction retourne le plateau du joueur

Lève:

TypeError – Si le paramètre n’est pas un dictionnaire (joueur)

Model.Joueur.detecter3verticalIA(plateau: list, couleur: int) → int

Fonction qui permet de détecter un alignement vertical de 3 pions de la même couleur dans un plateau de jeu.

Paramètres:

  • plateau – Plateau dans lequel on effectuer la détection

  • couleur – La couleur des pions pour lesquels on veut détecter l’alignement

Renvoie:

Le numéro de la colonne où l’alignement vertical de 3 pions est détecté, ou -1 s’il n’y a pas d’alignement
