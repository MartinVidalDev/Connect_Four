Model.Pion

Fonctions permettant de gérer les pions du jeu.

Exemple d’utilisation :

from Model.Constantes import *
from Model.Pion import construirePion

# Création d'un pion
pion = construirePion(const.ROUGE)
Model.Pion.construirePion(couleur: int) → dict

Fonction permettant de construire un pion

Paramètres:

couleur – Couleur du pion à construire

Renvoie:

Dictionnaire représentant un pion

Lève:

  • TypeError – Si le paramètre n’est pas un entier

  • ValueError – Si l’entier ne représente pas une couleur

Model.Pion.getCouleurPion(pion: dict) → int

Fonction permettant d’obtenir la couleur du pion

Paramètres:

pion – Pion dont ont récupère la couleur

Renvoie:

Entier représentant la couleur du pion passé en paramètre

Lève:

TypeError – Si le paramètre n’est pas un dictionnaire

Model.Pion.setCouleurPion(pion: dict, couleur: int) → None

Fonction permettant de changer la couleur du pion

Paramètres:

  • pion – Pion sur lequel on effectue la modification de couleur

  • couleur – Nouvelle couleur choisie pour le pion

Renvoie:

Ne retourne rien

Lève:

  • TypeError – Si le paramètre pion n’est pas un dictionnaire

  • TypeError – Si le paramètre couleur n’est pas un entier

  • ValueError – Si le paramètre couleur ne représente pas une couleur

Model.Pion.getIdPion(pion: dict) → int

Fonction permettant d’obtenir l’identifiant d’un pion

Paramètres:

pion – Pion sur lequel on cherche l’identifiant

Renvoie:

Entier représentant l’identifiant du pion

Lève:

TypeError – Si le paramètre n’est pas un dictionnaire

Model.Pion.setIdPion(pion: dict, val: int) → None

Fonction permettant de modifier l’identifiant d’un pion

Paramètres:

  • pion – Pion sur lequel on effectue la modification de l’identifiant

  • val – Nouvelle valeur de l’identifiant

Renvoie:

Ne retourne rien

Lève:

  • TypeError – Si le paramètre pion n’est pas un dictionnaire

  • TypeError – Si le paramètre valeur n’est pas un entier
